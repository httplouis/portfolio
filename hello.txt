Portfolio Website - Design & Development Prompt
Project Overview
A modern, dark-themed portfolio website with smooth animations, interactive elements, and a single-page layout. No sidebar navigation—sections flow vertically with smooth scrolling.
Design System
Color Palette
Background: #0a0a0a (primary), #0f0f0f (alternate sections), #1a1a1a (cards)
Text: #ffffff (primary), #ededed (foreground), #9ca3af (muted/gray-400)
Accent Colors:
Primary Blue: #3b82f6
Secondary Purple: #8b5cf6
Accent Pink: #ec4899
Borders: #2a2a2a
Gradient Text: Linear gradient from #3b82f6 → #8b5cf6 → #ec4899 with animated background position
Typography
Fonts: Inter, Poppins, system-ui sans-serif
Headings: Bold, letter-spacing -0.02em, line-height 1.2
Body: Line-height 1.6, relaxed spacing
Visual Effects
Gradient Text: Animated gradient shifting (5s infinite ease)
Glass Morphism: rgba(26, 26, 26, 0.7) with backdrop-filter: blur(10px)
Glow Effects: Radial gradients with color opacity (10-30%)
Particle Background: Animated particles in background
Spotlight Effect: Dynamic spotlight following mouse/scroll
Animated Gradient Overlay: Subtle animated gradient background

Animations & Interactions
Scroll Animations
Framer Motion: whileInView for scroll-triggered animations
Viewport: once: true, margin: "-100px"
Stagger: Children animate with delay (0.05s-0.1s intervals)
Transitions: Spring physics, duration 0.5-0.6s
Hover Effects
Scale: 1.05-1.1 on hover
Lift: y: -2 to y: -5 translate
Glow: Border color change, background gradient overlay
Rotate: Icons rotate 360° on hover (0.5s)
Page Load Animations
Container: Stagger children
Items: Fade in + slide up (opacity: 0, y: 20 → opacity: 1, y: 0)
Delays: Progressive delays for sequential appearance
Interactive Elements
Buttons: Scale on tap (whileTap: { scale: 0.95 })
Links: Color transitions, icon translations
Cards: 3D tilt, glow on hover
Form Inputs: Focus ring, border color change
